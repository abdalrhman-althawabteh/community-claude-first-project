# Leeds Alchemy — Landing Page

Official landing page for **Leeds Alchemy**, an AI & Automation Agency that builds custom AI systems, automation pipelines, and intelligent business applications.

---

## Live Preview

Open `index.html` directly in any browser — no server or installation required.

Or run a local server:

```bash
python3 -m http.server 3000
```

Then visit: `http://localhost:3000`

---

## Project Structure

```
├── index.html    # All page markup and content
├── style.css     # All styles, animations, and responsive layout
└── script.js     # Scroll animations, navbar behavior, counters
```

No frameworks. No build tools. No dependencies.
Pure HTML, CSS, and vanilla JavaScript.

---

## What's Inside

| Section | Description |
|---|---|
| **Navbar** | Sticky, frosted glass on scroll, mobile hamburger menu |
| **Hero** | Animated headline, 3 floating dashboard cards, gold grid background, cursor spotlight |
| **Marquee** | Scrolling ticker with key agency metrics |
| **Services** | Editorial numbered list — AI Solutions, Automation, Custom Apps |
| **Value Props** | 4-item grid with ghost number design elements |
| **Projects** | Asymmetric grid — AutoReply AI, InvoiceFlow, SalesOracle |
| **Not For Everyone** | Honest positioning — good fit vs. not a fit |
| **CTA** | Full-screen dark section with "ALCHEMY" background text |
| **Footer** | Clean minimal footer with nav links |
